# taskhub-csharp
O aplicație simplă și intuitivă pentru a servi ca un to-do list.
TaskHub este un dashboard de productivitate modern, construit în C#, care pune accent pe o experiență de utilizare (UX) fluidă și o ierarhie vizuală clară. Proiectul a fost dezvoltat pentru cursul de Proiectarea Interfețelor Utilizator.
Limbaj: C#
Funcții planificate:
*Vizualizare de ansamblu a progresului zilnic.
*Sortare automată a task-urilor în funcție de urgență.
*Suport nativ pentru teme vizuale.
*Interfața se adaptează fluid la redimensionarea ferestrei.
